思路:
`滑动窗口`：
代码：
```java
class Solution {
    public int lengthOfLongestSubstring(String s) {

        int max = 0;
        int len = s.length();
        if(len < 2) return len;
        int left = 0, right = 1;
        while(right < len){
            while(right < len && !(s.substring(left, right).contains(s.charAt(right) + ""))){
                right++;
            }
            if(right == len){
                max = Math.max(right - left, max);
            }
            max = Math.max(right - left, max);
            left += 1;
        }
        return max;
    }
}
```
