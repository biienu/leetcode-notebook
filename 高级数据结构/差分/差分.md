# 一维差分：

```java
原数组 arrs = {1,3,5,7};
差分数组 d = {1, 2, 2, 2};     //d[i] =  arrs[i] - arrs[i - 1];
//对原数组进行区间修改（或多次进行区间修改) 
[l, r] + v // 在[l, r] 区间的值 进行增加v，则只需对差分数组进行如下操作：
1. d[l] += v;  d[r + 1] -= v;
2. 对d进行前缀和，得到对[l, r]区间修改后的数组；
    
    
    
eg: [2,3] + 1;
d[2] += 1; d[4] -= 1;
d = {1,2,3,2};
sum-d = {1,3,6,8}; -> [2,3] + 1
```
